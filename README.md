# TfnValidationUI
A TFN validation UI which allows users to key in a TFN to validate it

# Steps
To run application:
1. npm install
2. npm start
3. Enter [http://localhost:3000/](http://localhost:3000/)
4. Ensure CORS is enabled in the browser to test

# Requirements
- NodeJS v10.16.3
